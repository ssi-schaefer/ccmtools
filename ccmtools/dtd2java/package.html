dtd2java is a DTD (Document Type Definition) to Java generator. It creates a model to store the data and a XML parser, which reads the XML file and returns the parse tree.

Usage: java ccmtools.dtd2java.Main DTD_file Java_package

The parameter Java_package is the Java name of the package for the model and the parser. Elements of UML namespaces will be stored in own sub-packages.

dtd2java uses the parser generator ANTLR.
Please put the ANTLR Jar-file into your classpath.

The generated XML parser only needs Java 1.4 !

Warning!
It is possible, that one '#PCDATA' element is split into multiple 'java.lang.String' elements, because the SAX-parser calls the method 'DTD_SAX_Handler.characters' too often!
