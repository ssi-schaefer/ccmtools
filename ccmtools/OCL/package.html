The OCL metamodel. There are three packages:
  • oclmetamodel - contains the core interfaces for the metamodel
    This package can be found in '$CCMTOOLS_HOME/MDR/mof/OCL.jar'.
  • ccmtools.OCL.utils - helper classes
  • ccmtools.OCL.parser - a parser for OCL text files
  • ccmtools.OCL.generators - code generators for OCL expressions
