UML to IDL and OCL converter.

The CORBA profile for UML defines a way to design CORBA interfaces and components by using the UML editor of your choice. uml2idl reads a XMI file and creates IDL and OCL files, which can be read by the code generators of the CCM tools.

uml2idl uses the XMI parser {@link uml_parser}.
This parser was created by {@link dtd2java}.
